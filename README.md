## 📖 Task 1 - Distribution Visualization

Visualized the distribution of Age (continuous variable) and Gender (categorical variable) from the Titanic dataset using a histogram and a bar chart.

**Key steps:**
- Loaded the dataset with Pandas
- Handled missing values in the Age column
- Plotted a histogram (with KDE) for Age distribution
- Plotted a bar chart for Gender distribution
